# Tetriz

Tetris for linux terminal.

How to run:  
Run this in a terminal:  
java -jar Tetriz.jar  

Controls:  
move pice left:              a  
move pice right:             d  
move pice down:              s  
rotate pice anti clock-wise: k  
rotate pice clock-wise:      l  

navigate menu: right and left arrows.
